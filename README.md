# Travel Booking Website

## Giới thiệu

Website đặt tour du lịch trực tuyến được xây dựng bằng PHP và MySQL

## Tính năng

### Khách hàng
- Đăng ký, đăng nhập
- Xem danh sách và chi tiết tour
- Tìm kiếm tour
- Đặt tour trực tuyến
- Thanh toán MoMo
- Xem lịch sử đặt tour
- Hủy booking
- Đánh giá tour
- Quản lý thông tin cá nhân

### Quản trị viên
- Quản lý tour
- Quản lý danh mục tour
- Quản lý khách hàng
- Quản lý booking
- Quản lý tin tức
- Thống kê doanh thu

## Công nghệ sử dụng

- PHP
- MySQL
- HTML/CSS
- JavaScript
- Bootstrap
- AJAX
- PHPMailer
- MoMo Payment Gateway
